# 0kb-status

Public-facing status page for the 0kb.io Lightning node.

This repository contains the static assets served at:

    https://0kb.io/status/

The page displays a structured view of the nodeâ€™s health, using a JSON snapshot generated by the private node-monitor service.

## Structure

```
    status/
        index.html      # Human-readable status page
        status.json     # Generated by node-monitor (ignored)

    .gitignore
    README.md
```

## Deployment

The contents of this repository are deployed to the web root used by nginx for the `/status/` location block.

Example nginx configuration:

```nginx
    location = /status {
        return 301 /status/;
    }

    location /status/ {
        alias /var/www/status/;
        autoindex off;
        add_header Cache-Control "no-store";
    }
```

## Notes

- `status.json` is generated by node-monitor and should not be committed.
- This repository contains no private operational details.
- All files are static and safe for public distribution.
